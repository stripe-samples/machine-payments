# Machine payments

This repository demonstrates how to implement machine-to-machine payments using protocols like x402.

## Prerequisites

- **Stripe account** with crypto payments enabled
- **Wallet** with testnet USDC (Base Sepolia)

## Getting Started

1. Clone this repository:
```bash
git clone https://github.com/stripe-samples/machine-payments
cd machine-payments
```

2. Follow the README in your chosen integration directory
